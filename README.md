MelodyStream - Lecteur de musique

[MelodyStream Screenshot](screenshot.png)

Une application web de lecteur de musique dédiée à l'album "Discovery" de Daft Punk, utilisant les données réelles de l'API Deezer.

À propos de l'album

*Discovery* est le deuxième album studio de Daft Punk, sorti le 7 mars 2001 chez Daft Life Ltd./ADA France.

- *Album ID*: 302127
- *Fans*: 326 507
- *Durée*: 61:02
- *Pistes*: 14 titres
- *Label*: Daft Life Ltd./ADA France
- *UPC*: 724384960650

Fonctionnalités

- Recherche de pistes dans l'album
- Écoute d'extraits de 30 secondes (liens Deezer officiels)
- Affichage des métadonnées détaillées (rank, durée, numéro de piste)
- Indication des contenus explicites
- Design responsive adapté à tous les écrans
- Interface utilisateur moderne et intuitive

Technologies utilisées

- HTML5
- CSS3 (Animations, Flexbox, Grid, Glassmorphism)
- JavaScript (ES6+)
- API Deezer (données réelles)
- Font Awesome Icons
- GitHub Pages
